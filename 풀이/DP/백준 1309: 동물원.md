3 X N 돌놓기 문제보다 간단한 2 X N 격자에 사자를 배치하는 문제

* 가능한 상태는 총 3가지이다.
  * 상태 0: 사자를 아무 곳에도 놓지 않음 -> 0,1,2 모든 열 가능
  * 상태 1: 사자를 첫 번째 줄에만 놓음 -> 0,2 열만 가능
  * 상태 2: 사자를 두 번째 줄에만 놓음 ->0,1 열만 가능

* dp[i][j]를 i번째 열이 j 상태일 때의 경우의 수라고 하면
  * dp[i][0] = (dp[i-1][0] + dp[i-1][1] + dp[i-1][2]) % 9901
  * dp[i][1] = (dp[i-1][0] + dp[i-1][2]) % 9901
  * dp[i][2] = (dp[i-1][0] + dp[i-1][1]) % 9901 가 된다.


```
import sys
input = sys.stdin.readline

n = int(input())

dp = [[0]*3 for _ in range(n+1)]

# 초기값 설정(첫 번째 줄)
dp[1][0] = 1
dp[1][1] = 1
dp[1][2] = 1

for i in range(2,n+1):
    dp[i][0] = (dp[i-1][0]+dp[i-1][1] + dp[i-1][2]) % 9901
    dp[i][1] = (dp[i-1][0] + dp[i-1][2]) % 9901
    dp[i][2] = (dp[i-1][0]+dp[i-1][1]) % 9901

print(sum(dp[n])%9901)
```
