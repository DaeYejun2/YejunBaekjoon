3 x N 벽을 2 x 1, 1 x 2 타일로 채우는 문제.

1. 사용하는 타일의 크기는 항상 2칸이므로 N이 홀수이면 답은 항상 0이다.
2. N = 2일때 3 x 2를 채우는 방법은 총 3가지 이다.
3. 예외 조각 (N = 4, 6, 8, ..일 때): N = 4 이상부터는 N = 2일 때의 조각들을 조합해서 만들 수 없는, 오직 그 길이에만 존재하는 고유한 패턴이 2개씩 발생한다.

### 점화식 세우기
dp[i]를 3 x i 벽을 채우는 방법의 수라고 하면
1. 가장 최근 2칸을 채우는 경우: dp[i-2] * 3 (N=2의 기본 조각 3개 적용)
2. 그보다 더 전의 예외 조각을 사용하는 경우:
   * dp[i-4] * 2 (N=4의 고유 조각 2개)
   * dp[i-6] * 2 (N=6의 고유 조각 2개)
   * ...
   * dp[0] * 2 (맨 처음까지 거슬러 올라감)
  

```
import sys
input = sys.stdin.readline

n = int(input())
dp = [0]*31
if n % 2:
    dp[n] = 0

dp[0] = 1
if n >= 2:
    dp[2] = 3

for i in range(4, n+1, 2):
    dp[i] = dp[i-2] * 3

    for j in range(i-4, -1, -2):
        dp[i] += dp[j]*2

print(dp[n])
```

* 직전 요소(i-2)에 x 3을 하는 이유: 3x2 크기의 '기본 조각' 3가지로 마지막을 채우는 경우. 모든 타일 채우기는 가장 기본적으로 이 2칸짜리 블록들의 조합으로 이루어지기 때문이다.
* 그 뒤의 모든 요소들(i-4, i-6,...,0)에 x2를 하는 이유: 더 작은 단위(2칸)로 쪼개지지 않는 예외 패턴이 나타나기 때문

* 마지막 덩어리가 2칸이면 dp[i-2] * 3 -> 기본 조각으로 마지막을 채우는 방법
* 마지막 덩어리가 4칸 이상(4, 6, 8, ..)이면 -> dp[남은 길이] x 2
