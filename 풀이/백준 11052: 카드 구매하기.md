### 점화식세우기
카드 i개가 포함된 카드팩의 가격을 P[i]라고 하고, 카드 n개를 갖기 위해 지불해야 하는 최대 금액을 DP[n]이라고 한다.

카드 n개를 만드는 마지막 단계는 다음 중 하나일 것이다.

* 카드 n-1개를 이미 맞췄고, 1개짜리 카드팩 P[1]을 추가로 삼
* 카드 n-2개를 이미 맞췄고, 2개짜리 카드팩 P[2]를 추가로 삼
* ...
* 카드 0개를 가졌고, n개짜리 카드팩 P[n]을 한 번에 삼

이를 일반화하면 다음과 같은 점화식이 나온다.

$$DP[i] = \max(P[k] + DP[i-k]) \quad (1 \le k \le i)$$

```
import sys
input = sys.stdin.readline

n = int(input())
data = [0]+list(map(int,input().split()))
dp = [0]*(n+1)

for i in range(1,n+1):
    for j in range(1,i+1):
        dp[i] = max(dp[i], data[j]+dp[i-j])

print(dp[n])
```
만약 카드팩 가격이 P = [0, 1, 5, 6, 7] 라면:

1. DP[1] = P[1] + DP[0] = 1
2. DP[2] = max(P[1]+DP[2-1] + P[2]+DP[2-2] = max(1+1, 5+0) = 5
3. DP[3] = max(P[1]+DP[3-1], P[2]+DP[3-2], P[1]+DP[3-3]) = max(1+5, 5+1, 6+0 ) = 6
4. DP[4] = max(P[1]+DP[4-1], P[2]+DP[4-2], P[3]+DP[4-3], P[4]+DP[4-4] = max(1+6, 5+5, 6+1, 7+0) = 10

최종적으로 카드 4개를 살 때 최대 비용은 10이다.
